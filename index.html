My Projects

Notable Projects

Detection of Energy-Inefficiency Patterns in Android Applications

Since android applications are continuously becoming increasingly complex, energy-related defects are of signiﬁcant importance to developers and app users. In this project, I am focusing on generating eﬀective test cases based on the control-ﬂow graph of the source code in order to detect the missing deactivation of energy-related resources by static and dynamic analyses.

Q-Maze

Solving any size of mazes with Q-learning algorithm. Diﬀerent reward functions are deﬁned and all of them solved the given maze in just one minute and maximum accuracy.

Dynamic Neural Network

Collaborating with Parsa Nooralinejad, a dynamic neural network with training and inference capability is implemented using python which its number of layers, type of activation functions, number of neurons in each layer, and loss function of the net could be customized.

Course Scheduling with Genetic algorithm

The list of class sections of all courses are given and a schedule is generated with the most student satisfaction using genetic algorithm.

AP Drive

Collaborating with Parsa Nooralinejad, we designed a routing algorithm to route packets on a given simulated network environment. In this project, several approaches are used for each routing request. The algorithm is designed to choose among several routes proposed by diﬀerent routing algorithms and setup the switching tables in such a way that the routing is the fastest possible path for the packet.

Designing a PID Controller for Three-Tank System

Taking advantage of Real-Time and Simhydraulics tools of MATLAB, I have designed a PID controller which specifies the level of water in each tank.
