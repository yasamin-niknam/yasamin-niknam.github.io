My Projects

Notable Projects

Vehicle Initialization and Counting

We developed a multi-phase model to stitch images, count vehicles, and generate heatmaps from a real image, which had been taken using a drone from a parking lot. Additionally, the annotated data was collected through the use of Amazon Mechanical Turk.

Multiple Object Tracking in crowded scenes

We developed a Tracktor from a detector model by exploiting the bounding box regression of an object detector in order to predict the position of an object in the next frame.

Bringing Old Photos Back to Life

We propose to restore old photos that suffer from severe degradation through a deep learning approach. Specifically, we train two variational autoencoders (VAEs) to respectively transform old photos and clean photos into two latent spaces. The translation between these two latent spaces is learned with synthetic paired data.
For the next stage, we will use this method to improve the quality of old videos.

Music retrieval

Implementating a system to recognize Persian Music using Machine Learning methods. For this project, many metrics have been tested and one have been chosen.

AP Drive

I have implemented a web platform called AP Drive which allows users to manage their files. This platform has been written in c++ with an object oriented approach with using of AP HTTP for the server side.
